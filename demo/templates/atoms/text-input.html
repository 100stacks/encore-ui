Text Input States

This page provides examples of the different text input states.

Plain Text Input

Plain text input without validation.

Text Input with Scroll (Incrementor/Decrementor)

Text input with value that can be entered than incremeneted and decremented with the scroll. The incrementor may also be utilized without a value.

Disabled Text Input

Disabled text input with value.

Text Input with Validtion

Text input validation can be provdied with active validation that checks each keystroke until the valid criteria is met, or it can check validation after an entry is made. Select/Deselect the checkbox to view the different behaviors.
