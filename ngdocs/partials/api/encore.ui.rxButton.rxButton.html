rxButton
directive in module encore.ui

Description

Renders a button which will disable when clicked and show a loading message and renable when operation is complete.

Usage

as element:
<rx-button
       loading-msg="{String}"
       default-msg="{String}"
       toggle="{Boolean}"
       disable="{Boolean}">
</rx-button>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
loadingMsgString
  • Text to be displayed when an operation is in progress.
defaultMsgString
  • Text to be displayed by default an no operation is in progress.
toggleBoolean
  • When true, the button will display the loading text.
disableBoolean
  • When true, the button will be disabled.
