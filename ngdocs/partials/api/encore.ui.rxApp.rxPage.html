rxPage
rxApp in module encore.ui

Description

Responsible for creating the HTML necessary for a page (including breadcrumbs and page title)

Usage

as element:
<rx-page
       [title]="{expression}"
       [subtitle]="{expression}">
</rx-page>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
[title]expression

Title of page

[subtitle]expression

Subtitle of page

Example

<rx-page title="'Page Title'"></rx-page>
