rxRelatedMenu
service in module encore.ui

Description

This directive is used to create a menu for a page. By default it shows on the left side, but can be configured for the direction you want.

Usage

as element:
<rx-related-menu
       collapsable="{Boolean=false}"
       default-state="{String='open'}"
       trigger="{String='click'}"
       state="{Boolean}"
       position="{String='left'}">
</rx-related-menu>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
collapsableBoolean=false
  • Is this menu going to be collapsable
default-stateString='open'
  • This option is only used if collapsable is true. Potential values are 'open' or 'closed'.
triggerString='click'
  • This option is only used if collapsable is true. Potential values are 'click' or 'hover'.
stateBoolean
  • Is the menu opened or closed.
positionString='left'
  • Where should the sidebar live? Possible values are 'left' or 'right'.
