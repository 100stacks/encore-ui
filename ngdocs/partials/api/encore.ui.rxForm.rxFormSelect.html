rxFormSelect
directive in module encore.ui

Description

This directive is used to create a <select> box w/options.

Usage

as element:
<rx-form-select
       options="{Array}"
       field-id="{String}"
       label="{String}"
       model="{Object}"
       required>
</rx-form-select>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
optionsArray
  • An array of options to generate. Example format:
    [{
  label: 'Staging',
  value: 'STAGING'
}, {
  label: 'ORD (Chicago)',
  value: 'ORD'
}];
fieldIdString
  • Used for label and input 'id' attribute
labelString
  • Text to use for <label>
modelObject
  • Value to bind input to using ng-model
requiredString
  • Value passed to select's 'ng-required' attribute
