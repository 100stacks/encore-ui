rxNotification
directive in module encore.ui

Description

Display a static message with styling taken from rx-notifications

Usage

as element:
<rx-notification
       type="{string}">
</rx-notification>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
typestring

The type of notification (e.g. 'warning', 'error')
