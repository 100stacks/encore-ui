rxFormFieldset
directive in module encore.ui

Description

This directive is used to wrap a set of input fields in the proper HTML

Usage

as element:
<rx-form-fieldset
       legend="{String}"
       description="{String}">
</rx-form-fieldset>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
legendString
  • Text to use for
descriptionString
  • Text to place below input
