rxFormOptionTable
directive in module encore.ui

Description

This directive is used to build a table of radio/checkbox inputs

Usage

as element:
<rx-form-option-table
       data="{Array}"
       columns="{array}"
       [selected]
       type="{String}"
       model="{Object}"
       field-id="{String}"
       required
       disable-fn="{Function}">
</rx-form-option-table>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
dataArray
  • Array of objects used to populate table. Properties must match columns data key. Example:
    [
  {
      'name': 'Option 1',
      'value': 0
  }, {
      'name': 'Option 2',
      'value': 1
  }, {
      'name': 'Option 3',
      'value': 2
  }
]
columnsarray
  • Array of objects with label/key values. Example:
    [{
  'label': 'Name',
  'key': 'name'
}]
selected
(optional)
String
  • Key of item that's selected (and therefore will have input disabled)
typeString
  • Type of input to be used
modelObject
  • Value to bind input to using ng-model
fieldIdString
  • Used for label and input 'id' attribute
requiredObject
  • Value passed to input's 'ng-required' attribute
disableFnFunction
  • Callback function to determine if option should be disabled. Takes tableId, fieldId, and rowId as parameters. Example:
    disable-fn="disableOption(tableId, fieldId, rowId)"
