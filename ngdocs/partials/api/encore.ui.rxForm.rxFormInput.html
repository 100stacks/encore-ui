rxFormInput
directive in module encore.ui

Description

This directive is used to create an <input> tag.

Usage

as element:
<rx-form-input
       type="{String}"
       required
       field-id="{String}"
       model="{Object}"
       min-length="{String}"
       max-length="{String}"
       max="{String}"
       min="{String}"
       label="{String}"
       suffix="{String}"
       description="{String}">
</rx-form-input>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
typeString
  • Value passed to input's 'type' attribute
requiredString
  • Value passed to input's 'ng-required' attribute
fieldIdString
  • Used for label and input 'id' attribute
modelObject
  • Value to bind input to using ng-model
minLengthString
  • Value passed to input's 'minLength' attribute
maxLengthString
  • Value passed to input's 'maxLength' attribute
maxString
  • Value passed to input's 'max' attribute
minString
  • Value passed to input's 'min' attribute
labelString
  • Text to use for
suffixString
  • Text to use after
descriptionString
  • Text to place after input
