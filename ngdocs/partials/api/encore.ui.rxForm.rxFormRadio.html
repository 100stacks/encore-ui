rxFormRadio
rxForm in module encore.ui

Description

This directive is used to create a set of <input type="radio"> tags.

Usage

as element:
<rx-form-radio
       options="{Array}"
       field-id="{String}"
       model="{Object}">
</rx-form-radio>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
optionsArray
  • An array of radio inputs to generate. Example format:
    [{
  label: 'Staging',
  value: 'STAGING'
}, {
  label: 'ORD (Chicago)',
  value: 'ORD'
}];
fieldIdString
  • Used for label and input 'id' attribute
modelObject
  • Value to bind input to using ng-model
