This component may be used to provide feedback on the progress of a workflow or action.

Encore Framework utilizes Angular Bootstrap progressbar. See the Angular Bootstrap page for more details.

  • There are two states for progress bars: active and complete. Active progress bars have animated stripes, which are produced by the class .active. Full progress bars do not have this class, and are solid and static in appearance.
  • Several options for displaying text with a progress bar are available: no text, a percentage (not italicized), and detailed text (italicized). Always try to show a percentage. Your focus should be to enter specific text, since it reveals both numeric progress and provides context for what progress is being tracked. See demo below.
