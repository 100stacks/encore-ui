This molecule is designed to be used in conjunction with other picker molecules to compose a valid ISO 8601 DateTime string in the format of YYYY-MM-DDTHH:mmZ.

Notice

  • This molecule will generate a String in the format of HH:mmZ to be used as the time portion of the ISO 8601 standard DateTime string mentioned above.
    • HH is the 24-hour format from 00 to 23
    • mm is the minutes from 00 to 59
    • Z is the UTC offset that matches [-+]\d{2}:\d{2}
  • This molecule will never generate anything other than a String.
