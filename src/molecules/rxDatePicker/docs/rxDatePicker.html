This molecule is designed to be used in conjunction with other picker molecules to compose a valid ISO 8601 DateTime string in the format of YYYY-MM-DDTHH:mm:ssZ.

Notice

  • This molecule will generate a String in the format of YYYY-MM-DD to be used as the date portion of the ISO 8601 standard DateTime string mentioned above.
  • This molecule will never generate anything other than a String.
